Général

Tout texte placé avant le mot clé "Glossaire" ou après le mot clé "Fin" n'est pas considéré comme faisant partie de l'algorithme.

Glossaire :

Le glossaire permet de définir toutes les variables qui vont être utilisées lors de l'algorithme.

Syntaxe : 
        Glossaire :
Expression régulière correspondante

"^glossaire\s?:?$"

Déclaration de variable :

Les variables doivent être déclarées de la manière suivante.

Syntaxe : 
        Type NomVariable Commentaire
Exemple : 
        Entier somme1 Cette variable permet de calculer la somme.
Expression régulière correspondante

"^(entier|r[ée]el|cha[îi]ne|caract[eè]re|bool[ée]en)\s+([a-z]+\w*)\s+((?:\w*\s*)*)$"

Début de l'algorithme :

Ce mot clé permet de définir le début de l'algorithme.

Syntaxe : 
        Début :
Expression régulière correspondante

"^d[eé]but\s?:?$"

Fin de l'algorithme :

Ce mot clé détermine la fin de l'algorithme.

Syntaxe : 
        Fin
Expression régulière correspondante

"^fin$"

Commentaire :

Ceci permet de placer des commentaires dans l'algorithme.

Syntaxe : 
        //
Exemple : 
        //Ceci est une ligne de commentaire
Expression régulière correspondante

"^//.*$"

Structures Conditionnelles

On entend par structure conditionnelle, la possibilité d'orienter un algorithme en fonction de choix particuliers. Les structures conditionnelles nous ouvrent de nouveaux horizons pour l'élaboration d'algorithmes. Cela représente une notion très importante qu'il faut pouvoir assimiler si l'on cherche à aller plus loin. Il en existe deux.

Si ... FinSi :

Elle permet d'exprimer une condition simple (binaire).

Syntaxe : 
        Si condition Alors
            Code_à_éxécuter_si_l'évaluation_de_la_condition_est_vraie
        FinSi
Exemple : 
        Si var_age < 18 Alors
            Afficher "Vous n'êtes pas encore majeur."
        FinSi
Expression régulière correspondante

"^si\s(" + expressionCompExpression de comparaison
Ex : variable1 < variable2 + ")\salors$"
"^finsi$"

Voir aussi :
Si ... Sinon... FinSi

Si ... Sinon... FinSi :

Elle permet d'exprimer une condition simple (binaire).

Syntaxe : 
        Si condition Alors
            Code_à_éxécuter_si_l'évaluation_de_la_condition_est_vraie
        Sinon
            Code_à_éxécuter_si_l'évaluation_de_la_condition_est_fausse
        FinSi
Exemple : 
        Si var_age < 18 Alors
            Afficher "Vous n'êtes pas encore majeur."
        Sinon
            Afficher "Vous pouvez conduire... OU boire!"
        FinSi
Expression régulière correspondante

"^si\s(" + expressionCompExpression de comparaison
Ex : variable1 < variable2 + ")\salors$"
"^sinon$"
"^finsi$"

Structures Répétitives

Dans certains algorithmes, on peut se retrouver à avoir besoin de faire plusieurs fois la même opération. Si elles ne sont pas nombreuses, on pourrait à la rigueur, dupliquer autant de fois que nécessaire le morceau de l'algorithme qui nous intéresse. Cependant, on se rend compte que cette solution n'est pas idéale. L'algorithme peut prendre rapidement une taille assez importante et si le traitement dupliqué est complexe et qu'une correction doit être effectuée, il faut alors la reporter sur chaque copie (sans compter que dans certains cas, on ne connaît pas à l'avance le nombre d'itérations que l'on aura à faire). Tout cela amène la notion de structure répétitive (ou boucle).

Tant Que :

Cette instruction évalue une condition et tant que celle-ci est vérifiée, les actions placées à l'interieur sont alors effectuées. Pour sortir de la boucle, il faut que la condition ne soit plus remplie.

Syntaxe : 
        TantQue (condition)
            action
        FinTantQue
Exemple : 
        tour = 5
        Afficher "Compte à rebours : "
        TantQue (tour > 0)
            Afficher tour
            tour = tour - 1
        FinTantQue
Expression régulière correspondante

"^tantque\s+(" + expressionCompExpression de comparaison
Ex : variable1 < variable2 + ")\s?$"
"^fintantque$"

Répéter Jusqu'à :

Cette instruction répète une action jusqu'à ce que la condition soit remplie.

Syntaxe : 
        Répéter
            action
        Jusqu'à (condition)
Exemple : 
        Répéter
            Afficher "Saisir un nombre positif : "
            Saisir nombre
        Jusqu'à (nombre > 0)
Expression régulière correspondante

"^r[ée]p[ée]ter$"
"^jusqu[\s']?[àa]\s?(" + expressionCompExpression de comparaison
Ex : variable1 < variable2 + ")\s?$"

Pour :

Elle affecte une valeur de départ à une variable, exécute les actions contenues dans la boucle, et incrémente la valeur de la variable jusqu'à la valeur de fin. Le pas d'incrémentation est par défaut de 1, mais il est possible de la régler avec l'instruction "pas de".

Syntaxe : 
        Pour variable1 de valeur1 à valeur2 [pas de valeur3]Optionnel faire :
            action
        FinPour
Exemple : 
        Pour i de 1 à 10 pas de 1 faire :
            Afficher "Compteur"
            Afficher i
        FinPour
	
        Pour i de 1 à 10 faire :
            Afficher "Compteur"
            Afficher i
        FinPour
Expression régulière correspondante

"^pour\s+(" + expressionVarVariable définie dans le glossaire
Ex : compteur + ")\s+de\s+(" + expressionValValeur
Ex : compteurMax
Ex : 12 + ")\s+[àa]\s+(" + expressionValValeur
Ex : compteurMax
Ex : 12 + ")\s+(\[pas\s+de\s+(" + expressionValValeur
Ex : compteurMax
Ex : 12 + ")\]\s+)?faire\s?:?\s?$"
"^finpour$"

Fonctions

Voici quelques fonctions d'entrée sortie.

Afficher :

Elle permet d'afficher une variable ou bien un texte dans la console d'exécution (Le texte doit être entre guillemets).

Syntaxe : 
        Afficher texte_ou_variable_à_afficher
Exemple : 
        Afficher "Bonjour"

        Afficher variable_de_la_mort_qui_tue
Expression régulière correspondante

"^afficher\s+(" + expressionValeur, ou texte, à afficher
Ex : compteur
Ex : "Voici un affichage" + "|\".*([^\"]*)\")\s?$"

Saisir :

Elle permet de saisir une variable.

Syntaxe : 
        Saisir variable
Exemple : 
        Saisir variable_age_de_lutilisateur
Expression régulière correspondante

"^saisir\s+(" + expressionVarVariable définie dans le glossaire
Ex : compteur + ")\s?$"
